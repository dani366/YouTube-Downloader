# 🚀 YouTube Downloader (MP3 & MP4)

Aplikasi ringan dan gratis untuk mendownload video dan audio dari YouTube secara instan. Versi Powefull bisa download sampai 4K TANPA BATAS 

## ✨ Fitur Utama
- Download MP3
- Download MP4 Mendukung resolusi hingga 4K tanpa batasan
- Indikator Real-Time Pantau persentase progres dan kecepatan unduhan langsung dari layar.

## 🛠️ Persyaratan Sistem (Prerequisites)
- Sistem Operasi: Windows 10 atau 11.
- Sudah disertakan aplikasi tambahan yt-dlp dan FFMPEG

## 📖 Cara Penggunaan
1. Download file `YouTube-Downloader.exe` dari menu **Releases**.
2. Simpan di folder mana saja (misalnya di Desktop).
3. Klik dua kali (Buka aplikasinya).
4. Paste link/tautan YouTube yang ingin diunduh ke kolom yang tersedia.
5. Pilih format yang diinginkan (Audio atau Video).
6. Klik **Mulai Download**. File akan otomatis tersimpan di folder yang sama dengan tempat Anda meletakkan aplikasi ini!
